# Motivation

## Problem

1. Need a GCC/G++ toolchain on Windows.
2. Prefer a Linux-like environment for development.

# Installation

Download the Ubuntu 22.04 for WSL:
```shell
https://cloud-images.ubuntu.com/releases/jammy/release/ubuntu-22.04-server-cloudimg-amd64-root.tar.xz
```
Install WSL and set up Ubuntu 22.04:
```shell
mkdir C:\WSL\Ubuntu-22.04
wsl --import Ubuntu-22.04 C:\WSL\Ubuntu-22.04 C:\Users\xayah\Downloads\ubuntu-22.04-server-cloudimg-amd64-root.tar.xz
wsl -d Ubuntu-22.04
```
