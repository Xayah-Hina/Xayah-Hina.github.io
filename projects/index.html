HinaFlow - Differentiable Fluid Simulator For Learning Tasks

HinaPE - Research oriented physics-based animation infrastructure.

2021/3 - 2024/5 Tags: Simulation Rendering Open in Github

HinaPE is a cross-platform and portable research-oriented physics-based animation infrastructure. HinaPE is designed with OpenGL & Vulkan backends.

Features:

  • Simulation: FLIP/APIC/PBF/DFSPH Fluid, FEM/XPBD/PD Cloth, etc.
  • Rendering: OpenGL/Vulkan based PBR Rasterizer / GLSL based Path Tracer, etc.
